Successful Alerts

Successful alerts will showcase buttons like:

  • Login

    login will be used to confirm a successful login alert.

  • Change Password

    This button will be used to change a users password.

  • Create Account

    This button is used to create a account and will alert when account is successfully created.

Warning Alerts

Warning alerts will showcase buttons like:

  • Close Session

    This button will close a session and will warn of session closer.

  • Send Email

    This button will send a email and will warn if there is no subject for the email.

  • Do not disturb mode

    This button will activate do not disturb mode and will warned for not recieving notifications.

Informativ Alerts

Informativ alerts will showcase buttons like:

  • Mute

    This button will mute the users microphone.

  • Cookie Recipe

    This button will display the average cookie recipe.

  • Show password

    This button will show the user their password.

Danger Alerts

Danger alerts will showcase buttons like:

  • System Check

    This button will check and warn of a potential virus'.

  • Open Email

    This button will open a email and warn of a potential danger of malware inside the email.

  • Create Account

    This button will create an account, if the password is too weak this alert will display.
